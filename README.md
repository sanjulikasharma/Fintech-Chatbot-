# Indian Unicorn Startup Chatbot

An AI-powered assistant designed to provide insights into India's 100+ unicorn startups using a RAG-based architecture.

## 🚀 Features
- **Contextual Awareness:** Remembers previous questions in the session.
- **Ambiguity Handling:** Asks for clarification rather than hallucinating.
- **Vector Search:** Uses FAISS for semantic matching of company backgrounds.
- **Observability:** Integrated with Langfuse for tracing and logging.
